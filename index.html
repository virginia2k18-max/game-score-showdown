How this works

Each player picked five games at the start of the year — one from each genre category — plus one alternate. Final tally adds the OpenCritic Top Critic Average of the five mains. The alternate only counts if a main pick is delayed past year's end.

Scores update automatically twice a month via the OpenCritic API. Hit reload to grab the freshest version.

Tier badges (Mighty / Strong / Fair / Weak) match OpenCritic's own verdicts.
